# 📝 To-Do List App com JDBC

Aplicação simples de lista de tarefas desenvolvida em Java, utilizando JDBC para conexão com banco de dados MySQL.

## ⚙️ Funcionalidades

- Criar conta
- Fazer login
- Adicionar tarefa
- Modificar tarefa
- Excluir tarefa
- Alterar senha
- Deletar conta

## 💾 Tecnologias

- Java
- JDBC
- MySQL
